An Analysis on Twitter News Accounts

This project intends to find the relationship between the level of user's engagement on a tweet and the countries mentioned in the tweet. It focuses on three Twitter news account. It is intended to be the final project for CS234 class. 

For a full write up of the project, visit the page: 
https://docs.google.com/document/d/1543taGCGI74xZLAHB_MLnGgG76W85SWMMjO6mDxziZE/edit?usp=sharing\

This project requires a few installations of the following libraries, including
* wbdata
* geopy
* geocoder
* Beautiful Soup
* requests
* numpy
* pandas
* pyplot

You can install them using the command:
pip install [library name]
